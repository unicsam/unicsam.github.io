Installation Guides
Running Portl on Windows
Windows SmartScreen may show a warning on first launch because the application is self-signed.
  1. Click "More info" on the dialog.
  2. Select "Run anyway" at the bottom.
Installing on Android (Direct APK)
  1. Download the APK file to your device.
  2. Tap the file. If prompted, allow installation from unknown sources.
  3. Follow the on-screen prompts to complete setup.
Running Portl on macOS
If macOS prevents launch due to unidentified developer verification:
  1. Open System Settings → Privacy & Security.
  2. Under the blocked app notification, click "Open Anyway".
Running on Linux (AppImage)
  1. Right-click the .AppImage file → Properties.
  2. Under Permissions, enable "Allow executing file as program".
  3. Or from terminal: chmod +x portl-linux.AppImage && ./portl-linux.AppImage
