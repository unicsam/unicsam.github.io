Installation Guides

Running Portl on Windows
Because this application is self-signed, Windows SmartScreen may display a warning dialog ("Windows protected your PC") on first launch.

To launch Portl:
  1. Click "More info" on the dialog popup.
  2. Select "Run anyway" at the bottom of the window.
Installing Portl on Android (Direct APK)
To install the direct APK package:
  1. Download the APK file onto your device.
  2. Tap the downloaded file. If prompted by your browser or system settings, allow installation from unknown sources.
  3. Proceed with the on-screen prompts to complete setup.
Running Portl on macOS
If macOS prevents launch due to unidentified developer verification:
  1. Navigate to System Settings > Privacy & Security.
  2. Scroll down to the Security section.
  3. Under the blocked app notification for Portl, click "Open Anyway".
Running Portl on Linux (AppImage)
To make the AppImage executable:
  1. Right-click the .AppImage file and select Properties.
  2. Under the Permissions tab, enable "Allow executing file as program".
  3. Launch the program directly, or initialize it from your terminal:
    chmod +x portl-linux.AppImage && ./portl-linux.AppImage
